Enterprise Operations Intelligence Platform

I built this project as an end-to-end analytics engineering and BI workflow, starting with raw CSV files and moving through cleaning, validation, database loading, and dbt-based transformation into a reporting-ready star schema. The project includes staging views, dimension and fact tables, data quality checks, and dashboard-oriented SQL for operational and product analysis.

Havi Hackathon - Demand Forecasting

This project was built for the HAVI / NIU Quick Service Restaurant Demand Forecasting Challenge, where the goal was to predict daily item demand for each restaurant and menu item in a future holdout window. The workflow combined exploratory analysis, time-based validation, forecasting features from history, calendar, weather, and promotions, and gradient boosting models designed around the competition's wMAPE objective.

Python LightGBM scikit-learn Forecasting Supply Chain Analytics

Ecommerce Strategy Analysis - A/B Testing

This project analyzes an ecommerce A/B test by cleaning experiment data, validating the control and treatment setup, and comparing conversion behavior and session engagement across groups and countries. The repository combines SQL-based analysis with Python data workflows, making it a practical example of how experimentation, data validation, and business interpretation come together in a decision-support project.
